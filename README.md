## Initial setup
1. If you don't have NodeJS, install it from [here](https://nodejs.org/)
2. Download this repository as ZIP (Code -> Download ZIP)
3. Extract the ZIP
4. Open a terminal in the extracted folder
5. Run `npm i` in order to install the dependencies
## Usage
Once you've done the initial setup, simply open a terminal in the destination folder and run `node index.js`
